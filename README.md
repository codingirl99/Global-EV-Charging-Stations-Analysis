# Global-EV-Charging-Stations-Analysis

Using Python and the dataset named Global EV Charging Stations by Vivek Attri posted on Kaggle.com, I have done an analysis on the data in Python using pandas and matplotlib libraries.

Below are some insights that are gathered from the project:

1) Charger Type (Pie Plot):
 > Insights: The dataset has an even split of types of chargers. There are three types of EV chargers: AC Level 1, AC Level 2, and DC Fast Charger. All have percentage of 33.3% which can be seen from the pie chart.
